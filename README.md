# NavCart - E-Commerce Platform

NavCart is a fully functional, lightweight frontend e-commerce website designed for seamless user shopping and advanced store management. Built purely with semantic HTML5, modern CSS3, and Vanilla JavaScript, it uses LocalStorage for client-side persistence.

## Key Features

* **User Shopping Flow:** Interactive product catalog, dynamic cart updates, wishlist functionality, and a simulated checkout pipeline.
* **Admin Dashboard:** Secure login portal to monitor real-time business statistics, low-stock alerts, and recent order logs.
* **Inventory Management (CRUD):** Fully operational interface to add, update, and delete products or suppliers dynamically.
* **Engagement Tools:** Features an automated hero banner slider, promotional countdown timers, and animated scroll effects.

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Database/State:** Browser LocalStorage
